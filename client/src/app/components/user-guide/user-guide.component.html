Guide d'utilisation

Crayon

Le crayon permet de dessiner un trait uni sur la surface de dessin

Pinceau

Le pinceau permet de dessiner un train d'une texture définie sur la surface de dessin

Rectangle

Le rectangle permet de dessiner un rectangle ou un carré en pesant sur la touche shift

Créer un dessin

En cliquant sur le bouton "Créer un dessin ", vous ouvrirez un nouvel éditeur dans le quel vous pourrez laisser aller votre créativité!
