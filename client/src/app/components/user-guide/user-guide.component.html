Créer un dessin

En cliquant sur le bouton "Créer un dessin ", vous ouvrirez un nouvel éditeur dans le quel vous pourrez laisser aller votre créativité!

Continuer un dessin

En cliquant sur le bouton "Continuer un dessin ", vous ouvrirez le caroussel de dessins et vous pourrez choisir un dessin à continuer!

Ouvrir le caroussel de dessins

En cliquant sur le bouton "Ouvrir le caroussel de dessins ", vous ouvrirez le caroussel de dessins et vous pourrez choisir un dessin sauvegardé afin de le continuer, le modifier ou le supprimer!
